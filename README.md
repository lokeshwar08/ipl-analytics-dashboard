
# IPL Analytics Project

## Overview
This project is a cloud-based IPL analytics dashboard using:
- AWS S3
- AWS EC2
- MySQL/PostgreSQL
- Python
- Tableau

## Features
- Cricket data analysis
- SQL-based player statistics
- Tableau dashboards
- Flask web app
- Match winner prediction ready structure

## Project Structure
- datasets/ -> Raw IPL datasets
- sql_queries/ -> SQL scripts
- python_scripts/ -> Data cleaning and ETL
- tableau_dashboard/ -> Tableau workbook storage
- app/ -> Flask application

## Technologies
- Python
- Pandas
- Flask
- SQL
- Tableau
- AWS

## Run Project
1. Install requirements
2. Run Flask app
3. Execute SQL scripts
4. Connect Tableau to SQL database
